# IA Analizadora de Clientes – Proyecto HPE

Sistema de inteligencia artificial para detectar posibles clientes,
evaluar su interés y recomendar productos adecuados de forma automática.

-------------------

## Objetivo

Crear un modelo capaz de:

- Analizar datos de mercado
- Detectar señales de compra
- Calificar clientes potenciales
- Recomendar productos
- Generar reportes automáticos

------------------------

## Arquitectura del sistema

1. Recolector de datos
2. Extractor de señales
3. Scoring de clientes
4. Clasificador
5. Recomendador HPE
6. Generador de speech
7. Sistema de salida/reportes
